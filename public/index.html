HTML <html> 요소는 HTML 문서의 루트(최상단 요소)를 나타내며, "루트 요소"라고도 부릅니다. 모든 다른 요소는 <html> 요소의 후손이어야 합니다. ( 참고 )

이미지와 피규어

학습 주제

  • HTML
  • CSS
  • ECMAScript (JavaScript)
  • Node.js
  • Webpack
  • React
  • Next.js

학습 방법

온라인 화상 강의의 특성 상, 현장 강의를 수강하는 것과는 학습 환경이 다릅니다.

강의는 각 주제의 개념을 설명 함과 동시에 라이브 코딩을 진행합니다. 그러므로 배움과 익힘의 시간을 병행하되, 강의에서 다룬 내용을 잘 정리한 다음, 추후에 시간을 내서 다시 실습하며 익힐 것을 권장합니다.
